Component

Animatext

Buttons

  • New EffectClick it to add a new effect in the first place.

  • DocumentationClick it to view the documentation.

  • Settings Click it to enter or exit system settings.

Settings

  • Retained TextThe retained text content in the text component after the Animatext component parsing.

    • Input textThe text initially entered of the text component.

    • Parsed textThe parsed text of the Animatext component.

    • Output textThe parsed text of the text component.

    • Effect textThe text containing the effect.

    • Blank textNo text included.

    • When the retained text is undone in editor playing mode, the text for effect execution is the text in the component prior to the undoing instead of the text input prior to the undoing.

    Note: In UGUI version, it is recommended to select "blank text" as it helps reduce the performance cost of generating original vertices.

  • Disabled TextThe retained text content in the text component when the Animatext component is disabled.

    • Input textThe text initially entered of the text component.

    • Parsed textThe parsed text of the Animatext component.

    • Output textThe parsed text of the text component.

    • Effect textThe text containing the effect.

    • Blank textNo text included.

    Note: In TextMeshPro version, the text style of the text component is reset if the value is not input text.

  • Disabled Effects The way effects are processed when the Animatext component is disabled.

    • StopStops all the effects in the Animatext component.

    • RefreshRefreshes all the effects in the Animatext component.

    • ClearClears all the effects in the Animatext component.

    • RemainRetains all the effects in the Animatext component.

  • Tag Symbols

    • Angle bracketsUsing angle brackets "<" and ">" to denote tag symbols.

    • Round bracketsUsing round brackets "(" and ")" to denote tag symbols.

    • Square bracketsUsing square brackets "[" and "]" to denote tag symbols.

    • Curly bracketsUsing curly brackets "{" and "}" to denote tag symbols.

  • Marker Symbols

    • SlashesUsing the slashes "/" to denote marker symbols.

    • BackslashesUsing the backslashes "\" to denote marker symbols.

    • VerticalBarsUsing the vertical bars "|" to denote marker symbols.

  • Scale SDFThe scale of SDF. (TextMeshPro only)

Menu

  • Log Origin TextLog the origin text of the Animatext component.

    • In special cases, the input text is different from the displayed text in the text component. You can use this method to get the original parsed text in the Animatext component.

AnimatextEventHelper

Buttons

  • New EffectClick it to add an event helper in the first place.

  • DocumentationClick it to view the documentation.

Event Helper

  • AnimatextThe Animatext component of the effects. If the value of "Animatext" is null, it will automatically get the Animatext component of the current game object at runtime.

  • Index ListThe index list of the effects in the Animatext component. If the index list contains -1, it means the index list contains all the indexes of the effects in the Animatext component.

  • EventsThe events of the effects.

    • onEndEventThe event added to Effect.onEnd.

    • onPauseEventThe event added to Effect.onPause.

    • onPlayEventThe event added to Effect.onPlay.

    • onProceedEventThe event added to Effect.onProceed.

    • onStartEventThe event added to Effect.onStart.

    • onStopEventThe event added to Effect.onStop.
