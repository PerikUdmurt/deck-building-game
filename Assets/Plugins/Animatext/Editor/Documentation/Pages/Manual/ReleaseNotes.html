Release Notes

V1.2.4 (Current)

  • Fixed incorrect effect state after setting new text when the "Effect.AutoPlay" value is false.

  • Now all effects will be refreshed immediately after modifying the text.

V1.2.3 (2024-04-17)

  • Fixed compatibility issue with TextMeshPro in Unity 2023.2.

V1.2.2 (2024-03-12)

  • Fixed text disappearing when the Animatext editor path is not initialized.

V1.2.1 (2024-02-06)

  • Fixed text disappearing when continuously enabling or disabling the Animatext component.

V1.2.0 (2024-01-19)

  • Added new component: AnimatextEventHelper.

  • Added new "Effect.AutoEnd" property to control whether the effect will end automatically.

  • Added new "Effect.AutoStop" property to control whether the effect will stop automatically.

  • Fixed issue where "Effect.OnPlay" event cannot be triggered after automatic start.

  • Fixed issue when calling "BaseAnimatext.Refresh" method on inactive game objects.

  • Fixed incorrect text animation when the "PingpongLoop" value is true and the "LoopCount" value is even number in the preset.

V1.1.0 (2022-02-06)

  • Added 21 text effects: CFadeA03, CStepA01-A04, CCustomA01-A05, TFadeA03, TStepA01-A05, TCustomA01-A05.

  • Added 13 color modes: ColorBurn, ColorDodge, Darken, Divide, Exclusion, HardLight, Lighten, LinearLight, Normal, Overlay, Screen, SoftLight, Subtract.

  • Added locked status for the Animatext component.

  • Now the execution order of the Animatext component is 0.

  • Fixed incorrect text animation when the Animatext component was enabled again.

  • Removed 4 text effects: CBasicA05, CElasticA05, CBounceA05, CWaveA05.

V1.0.2 (2021-12-03)

  • Added new integrations: PlayMaker, Quest Machine.

  • Fixed incorrect text during text update in the Animatext component.

  • Fixed incorrect effect state when the Animatext component is refreshed.

V1.0.1 (2021-11-05)

  • Added effect previews to the documentation.

  • Improved word parsing and line parsing in the "Textinfo" class.

  • Fixed incorrect text animation when front interval of execution effect is less than 0.

  • This is the firsr version.
